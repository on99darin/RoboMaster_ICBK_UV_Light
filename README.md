## 依赖工具及软件环境
`工具:`嘉立创EDA（专业版）

`软件环境:`Windows10

## 食用须知：
UV灯板必须使用散热鳍片，规格：`20mm*14mm*6mm`或定制

## 测试
接在发射机构的分电板上连续发射测试时长超过30分钟
上电连续测试60分钟，稳定使用
